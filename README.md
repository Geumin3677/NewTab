# NewTab
Chrome extension을 이용한 새 탭 페이지 커스텀

![image](https://user-images.githubusercontent.com/91381965/222915109-db0974c5-4d5c-4173-95a7-61bd16e746f8.png)

## 기능
+ 현재 시각표시
+ 학교 시간표 표시
+ 렌덤 아이유 배경화면
+ 렌덤 아이유 명가사
